Semantic markup
  • It is the markup that relates to the meaning of content
  • There are two types of tags 1)Semantic 2)Non-semantic
  • Benifit of semantic markup is : it provides a structured and meaningful layout to the webpage
    (2) makes website more SEO friendly oriented
    (3) Readable and benificial for screen readers so, improves UX
  • Semantic-tags: on reading these tags we can guess about meaning of contents
  • Non-semantic tags : on reading these tags we can't say anything about meaning of content inside the tag
  • Examples are:
    • paragraph
    • heading tag
    • header
    • footer
    • main
    • article
    • section( Works same as div and span )
    • aside ( in this we keep all indirectly related to content )
